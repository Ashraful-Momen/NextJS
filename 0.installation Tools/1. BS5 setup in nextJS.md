
---

## **Step 1: Create a Next.js project (if you don’t have one)**

```bash
npx create-next-app@latest my-nextjs-app
cd my-nextjs-app
npm install
```

---

## **Step 2: Install Bootstrap 5**

You can install via **npm**:

```bash
npm install bootstrap
```

---

## **Step 3: Import Bootstrap CSS**

### **Option 1: Import globally in `_app.js`**

Edit `pages/_app.js`:

```javascript
import 'bootstrap/dist/css/bootstrap.min.css';
import '../styles/globals.css'; // your global styles

function MyApp({ Component, pageProps }) {
  return <Component {...pageProps} />;
}

export default MyApp;
```

This ensures that Bootstrap’s CSS is available across your entire Next.js app.

---

### **Option 2: Import in a specific component (less common)**

```javascript
import 'bootstrap/dist/css/bootstrap.min.css';

export default function Home() {
  return <h1 className="text-primary">Hello Bootstrap</h1>;
}
```

---

## **Step 4 (Optional): Add Bootstrap JS functionality**

Bootstrap JS requires **Popper.js**, but if you want **React-friendly components**, it’s better to use `react-bootstrap` instead.

### **Option A: Using Bootstrap JS directly**

```bash
npm install @popperjs/core
```

Then import JS in `_app.js`:

```javascript
import 'bootstrap/dist/css/bootstrap.min.css';
import 'bootstrap/dist/js/bootstrap.bundle.min.js';
```

**Note:** Using raw Bootstrap JS in Next.js may cause hydration errors in some cases, especially with SSR.

---

### **Option B (Recommended): Use React Bootstrap**

```bash
npm install react-bootstrap bootstrap
```

Example usage:

```javascript
import { Button } from 'react-bootstrap';

export default function Home() {
  return <Button variant="primary">Click Me</Button>;
}
```

This avoids SSR issues because components are React-ready.

---

## **Step 5: Test it**

Add a Bootstrap component:

```javascript
export default function Home() {
  return (
    <div className="container mt-5">
      <h1 className="text-center text-primary">Bootstrap 5 with Next.js</h1>
      <button className="btn btn-success">Click Me</button>
    </div>
  );
}
```

Run the app:

```bash
npm run dev
```

Open `http://localhost:3000` to see Bootstrap styling in action.

---

✅ **Summary:**

* Use `npm install bootstrap` for CSS only.
* Import `bootstrap/dist/css/bootstrap.min.css` in `_app.js`.
* For JS components, either use `bootstrap/dist/js/bootstrap.bundle.min.js` **or** use `react-bootstrap` (recommended for Next.js).

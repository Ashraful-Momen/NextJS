# **Tutorial: Next.js App Router – User Dashboard & User Details**

### **1️⃣ Folder Structure**

```
app/
└─ dashboard/
   ├─ user/
   │  ├─ page.jsx       // User Dashboard
   │  └─ [id]/
   │      └─ page.jsx   // User Details
   └─ page.jsx          // Optional: dashboard main page
```

---

### **2️⃣ User Dashboard Page**

File: `app/dashboard/user/page.jsx`

```tsx
import Link from 'next/link';
import React from 'react';

const UserDashboard = () => {
  return (
    <div>
      <h1>User Dashboard Page</h1>
      <ul>
        <li><Link href="/dashboard/user/1">User Details 1</Link></li>
        <li><Link href="/dashboard/user/2">User Details 2</Link></li>
        <li><Link href="/dashboard/user/3">User Details 3</Link></li>
      </ul>
    </div>
  );
};

export default UserDashboard;
```

* This lists **users** and links to their detail pages.

---

### **3️⃣ User Details Page**

File: `app/dashboard/user/[id]/page.jsx`

```tsx
import React from 'react';

// Server Component: params is async in Next.js 16+
const UserDetails = async ({ params }) => {
  const { id } = await params; // unwrap async params

  return (
    <div>
      <h1>User Details Page</h1>
      <p>User ID: {id}</p>
    </div>
  );
};

export default UserDetails;
```

✅ This will display the ID from the URL.

---

### **4️⃣ Test it**

1. Run the dev server:

```bash
npm run dev
```

2. Open the dashboard:

```
http://localhost:3000/dashboard/user
```

3. Click any user link:

```
http://localhost:3000/dashboard/user/1
http://localhost:3000/dashboard/user/2
http://localhost:3000/dashboard/user/3
```

* Each page shows **User Details Page** with the corresponding **ID**.

---

### **5️⃣ Notes**

1. `[id]` → dynamic route parameter.
2. `params` in **Next.js 16+ server components** is **async** → `await params`.
3. No `'use client'` is needed unless you want hooks/state.
4. Use `<Link>` for client-side navigation.

---

### **6️⃣ Optional: Add Back Link**

```tsx
import Link from 'next/link';

const UserDetails = async ({ params }) => {
  const { id } = await params;

  return (
    <div>
      <h1>User Details Page</h1>
      <p>User ID: {id}</p>
      <Link href="/dashboard/user">Back to Dashboard</Link>
    </div>
  );
};
```

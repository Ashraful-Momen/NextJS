## 🔄 1. **SWR (Client-Side Data Fetching)**

Use **SWR** when you need **real-time, dynamic data** on the client (e.g., cart, user profile, live inventory).

### ✅ Install SWR
```bash
npm install swr
```

### ✅ Example: Fetch Product in Client Component
```jsx
// app/product/page.js (Client Component)
'use client';

import useSWR from 'swr';

const fetcher = (url) => fetch(url).then(res => res.json());

export default function ProductPage() {
  const { data, error, isLoading } = useSWR('/api/product/123', fetcher);

  if (isLoading) return <div>Loading...</div>;
  if (error) return <div>Error!</div>;

  return <h1>{data?.name}</h1>;
}
```

> ✅ **SWR = Stale-While-Revalidate**  
> - Shows stale data first  
> - Fetches fresh data in background  
> - Auto-revalidates on focus/reconnect

## 🕒 2. **ISR (Incremental Static Regeneration)**

Use **ISR** for **public pages that change occasionally** (e.g., product listings, blog posts).

### ✅ Example: Product Page with ISR
```jsx
// app/products/[id]/page.js (Server Component)
export async function generateStaticParams() {
  // Pre-render these at build time
  return [{ id: '1' }, { id: '2' }];
}

export async function generateMetadata({ params }) {
  const res = await fetch(`http://laravel/api/shop/products/${params.id}`);
  const product = await res.json();
  return { title: product.data.name };
}

// ✅ ISR: Revalidate every 1 hour
export const revalidate = 3600;

export default async function ProductPage({ params }) {
  const res = await fetch(`http://laravel/api/shop/products/${params.id}`);
  const product = await res.json();

  return <h1>{product.data.name}</h1>;
}
```

> ✅ **How ISR works**:  
> - Page is **statically generated** at build time (or on first request)  
> - Served instantly from disk/CDN  
> - Every **3600 seconds**, Next.js regenerates it in the background  
> - Users always see fast static HTML

---

## 🆚 When to Use What?

| Use Case | Use |
|--------|-----|
| Product detail page (public) | ✅ **ISR** (`revalidate`) |
| Cart, user dashboard, order history | ✅ **SWR** (client-side) |
| Homepage, category pages | ✅ **ISR** |
| Real-time stock count | ✅ **SWR** + polling |

## 💡 Pro Tip: Combine Both!

```jsx
// Product page: static HTML (ISR) + live stock (SWR)
export const revalidate = 3600;

export default async function ProductPage({ params }) {
  const res = await fetch(`.../products/${params.id}`);
  const product = await res.json();

  return (
    <>
      <h1>{product.data.name}</h1>
      {/* Static price, description */}
      
      {/* Live stock (client-side) */}
      <LiveStock productId={params.id} />
    </>
  );
}

function LiveStock({ productId }) {
  const { data } = useSWR(`/api/stock/${productId}`, fetcher);
  return <p>Stock: {data?.quantity}</p>;
}
```

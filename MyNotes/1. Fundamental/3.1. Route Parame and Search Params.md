# рЯФА 2. Dynamic Route (`[param]`)

### Example: `/blog/:slug`

```jsx id="y5x2am"

рЯФН params Explanation:

- "params" contains dynamic values from the URL.
```

# рЯФО 4. Query Params (`searchParams`)

### Example: `/blog?category=tech&page=2`

```jsx id="b7n3lp"

/*
рЯФН searchParams Explanation:

- Used for query string (?key=value)
- Automatically available in page component
*/

  return (
    <div>
      <p>Category: {searchParams.category}</p>
      <p>Page: {searchParams.page}</p>
    </div>
  )
}

/*
URL:
"/blog?category=tech&page=2"

searchParams = {
  category: "tech",
  page: "2"
}
*/
```

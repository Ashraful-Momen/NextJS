ReferenceError: document is not defined

### **How to fix it**

You **cannot import Bootstrap JS globally in server components**. You have 3 options:

---

## **Option 1: Only use Bootstrap CSS**

Remove the JS import:

```ts
import 'bootstrap/dist/css/bootstrap.min.css';
// DO NOT import JS here
```

✅ Works fine if you only need **CSS styling**.

---

## **Option 2: Dynamically load Bootstrap JS in client component**

Use **Next.js dynamic import** to load JS **only on the client**.

```tsx
'use client';

import { useEffect } from 'react';
import 'bootstrap/dist/css/bootstrap.min.css';

export default function BootstrapWrapper() {
  useEffect(() => {
    import('bootstrap/dist/js/bootstrap.bundle.min.js');
  }, []);

  return (
    <div className="container mt-5">
      <h1 className="text-primary text-center">Bootstrap 5 + Next.js</h1>
      <button className="btn btn-success" data-bs-toggle="modal" data-bs-target="#myModal">
        Open Modal
      </button>

      {/* Example modal */}
      <div className="modal fade" id="myModal" tabIndex={-1}>
        <div className="modal-dialog">
          <div className="modal-content">
            <div className="modal-header">
              <h5 className="modal-title">Modal Title</h5>
              <button type="button" className="btn-close" data-bs-dismiss="modal"></button>
            </div>
            <div className="modal-body">Hello Bootstrap JS!</div>
          </div>
        </div>
      </div>
    </div>
  );
}
```

> Key: `useEffect` ensures JS only runs **on the client**, avoiding SSR errors.

---

## **Option 3 (Recommended for Next.js): Use `react-bootstrap`**

Instead of loading Bootstrap JS directly, use React components:

```bash
npm install react-bootstrap bootstrap
```

Example:

```tsx
'use client';

import 'bootstrap/dist/css/bootstrap.min.css';
import { Button, Modal } from 'react-bootstrap';
import { useState } from 'react';

export default function Home() {
  const [show, setShow] = useState(false);

  return (
    <div className="container mt-5">
      <Button variant="primary" onClick={() => setShow(true)}>
        Open Modal
      </Button>

      <Modal show={show} onHide={() => setShow(false)}>
        <Modal.Header closeButton>
          <Modal.Title>Bootstrap Modal</Modal.Title>
        </Modal.Header>
        <Modal.Body>Hello Bootstrap JS via React!</Modal.Body>
      </Modal>
    </Modal>
  );
}
```

✅ This **avoids SSR issues completely**.

---

So the **main problem** is:

> You cannot import `bootstrap.bundle.min.js` in a server-side component (`app/layout.tsx`).

Either **use CSS only**, **load JS dynamically in a client component**, or **use react-bootstrap**.

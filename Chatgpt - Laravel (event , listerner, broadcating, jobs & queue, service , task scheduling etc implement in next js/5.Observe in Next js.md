# âœ… **1. Best & simplest solution â†’ Node.js EventEmitter**

Works exactly like Laravel observers.

### **Create an Observer System**

### **A. Create `events/observer.js`**

```javascript
import { EventEmitter } from "events";

const observer = new EventEmitter();

// limit listeners so it doesnâ€™t leak
observer.setMaxListeners(20);

export default observer;
```

---

### **B. Trigger events whenever something happens**

Example API route: `/api/user/create`

```javascript
// pages/api/user/create.js
import observer from "../../../events/observer";

export default function handler(req, res) {
  const { name, email } = req.body;

  // DB insert logic....
  const user = { id: 1, name, email };

  // Fire observer event
  observer.emit("user_created", user);

  res.json({ success: true, user });
}
```

---

### **C. Add Observer Listeners (like Laravel observer)**

```javascript
// observers/userObserver.js
import observer from "../events/observer";

observer.on("user_created", (user) => {
  console.log("User Created Observer Fired:", user);

  // Example extra logic:
  // send email
  // push notification
  // log activity
  // schedule welcome job
});

observer.on("user_updated", (user) => {
  console.log("User Updated:", user);
});
```

Then import it in `server.js` or `next.config.js` startup file:

```javascript
import "./observers/userObserver";
```

Now the observer is active.

# âœ… **2. Observer using Prisma (Database hooks)**

If your Next.js uses Prisma ORM:

```javascript
  }

  return next(params);
});
```

This is very close to Laravel model observers.
